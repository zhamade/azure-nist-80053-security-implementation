# azure-nist-80053-security-implementation
Implementation of NIST SP 800-53 security controls in Microsoft Azure using Entra ID, Azure Policy, RBAC, Key Vault, Microsoft Defender for Cloud, and Azure Monitor.

Azure NIST SP 800-53 Security Implementation
Overview

This project demonstrates the implementation of Azure security controls aligned with the NIST SP 800-53 Rev. 5 cybersecurity framework. The objective was to design and secure a cloud environment using Microsoft Azure services while applying identity management, governance, auditing, and security monitoring best practices.

Technologies Used:

Microsoft Azure

Microsoft Entra ID

Azure RBAC

Azure Policy

Azure Key Vault

Microsoft Defender for Cloud

Azure Monitor

Azure Resource Groups

Virtual Machines

Storage Accounts

Objectives:

Implement least privilege using Azure RBAC

Configure Microsoft Entra ID users and groups

Apply Azure Policy governance

Secure secrets using Azure Key Vault

Monitor resources using Defender for Cloud

Capture administrative activity using Azure Monitor

Map Azure controls to NIST SP 800-53
